Tested with Firefox 41.0 and NVDA 2015.3 on Windows 7, to see how iframes behave when accessed.

Down arrow (browse mode)

When accessing the iframe via the down arrow, NVDA reads: iframe banner landmark navigation landmark button open menu

M (browse mode)

When accessing the iframe via the M key, NVDA reads: This is the iframe title attribute iframe banner landmark navigation landmark open menu button

Tab (browse mode)

Tab goes to the iframe container and no content is read. Pressing Tab again moves focus to the first focusable element and NVDA reads: This is the iframe title attribute iframe banner landmark navigation landmark open menu button

Tab (forms mode)

Tab goes to the iframe container and NVDA reads: This is the iframe title attribute iframe this is the iframe title attribute document.
