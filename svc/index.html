HTML and CSS: Level 1 @ SVC

Schedule and materials for an entry level HTML and CSS class at the School of Visual Concepts in Seattle, for January 7 - February 4, 2014.

Got questions? Email me at dep@dpersing.com.

Class Schedule

Week 1: January 7

  • Introduction to HTML
  • Introduction to pre-coding design and planning concepts
  • Basic HTML elements and hierarchy
  • HTML file structure and organization
  • Introduction to using text editors
  • Storing and serving HTML and CSS on Dropbox

Week 2: January 14

  • Preparing images for the web
  • Block versus inline HTML elements
  • Introduction to CSS
  • Styling basic HTML elements
  • Understanding the style cascade
  • Validating HTML and CSS

Week 3: January 21

  • Styling backgrounds
  • Using ids and classes with CSS
  • CSS box model
  • Resetting default browser styles
  • CSS abbreviations

Week 4: January 28

  • More HTML5 container elements
  • Inline-block elements
  • Multi-column and grid layouts
  • CSS pseudo-classes
  • HTML data tables

Week 5: February 4

  • Lingering questions
  • iFrames
  • Web fonts
  • Overview of Javascript
  • Overview of CSS3
  • Overview of responsive and mobile-first design
  • Overview of web accessibility
  • Overview of version control
