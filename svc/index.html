HTML & CSS: Level 1 Winter 2014

An introductory course on HTML & CSS principles and practice, taught by Devon Persing for the School of Visual Concepts in Seattle.

Class schedule

Week 1: Intro to coding and HTML November 4

  • Planning web pages
  • Tools for writing code
  • Intro to HTML documents and elements

Other topics: Introductions, managing and naming your files

Slides (.pdf) Start files (.zip) End files (.zip)

Week 2: Intro to CSS November 11

  • Block and inline elements
  • CSS for fonts and colors
  • The style cascade

Other topics: Separation of content and style, prepping web-friendly images, validating your code

Slides (.pdf) Start files (.zip) End files (.zip)

Week 3: Better CSS and HTML5 containers November 18

  • More kinds of selectors for style
  • More background styles
  • HTML5 semantic container elements
Other topics: Web fonts, favorite Thanksgiving side dishes

Don't forget: There's no class during the week of Thanksgiving! We'll resume December sessions in SVC's new location.

Week 4: The CSS box model and intro to layouts December 2

  • CSS abbreviations
  • Understanding and manipulating the CSS box model
  • Using CSS for positioning content
  • Resetting browser default styles

Other topics: HTML data tables

Week 5: More complex layouts, embedded media, and related concepts December 9

  • Mobile-first thinking
  • Responsive/flexible layouts
  • iframes and HTML5 media

Other topics: Accessibility and HTML/CSS, how Javascript works with HTML and CSS, code libraries and other tools, your lingering questions
